# Spring Boot Student API

This project is a simple REST API built using Spring Boot.

The API allows users to retrieve and add student records.

## Features

- Get all students
- Add new student
- REST API endpoints
- JSON request and response

## Technologies Used

- Java
- Spring Boot
- REST API
- Maven

## Endpoints

GET /students

Returns a list of students.

POST /students

Adds a new student to the system.
